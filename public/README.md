# Public assets

Place your static files here. Vite serves files in this directory at the site root.

To enable the resume download button in `src/components/Hero.tsx`, add your resume as:

- public/resume.pdf

Then access it at:

- /resume.pdf

Tip: Keep the filename exactly `resume.pdf` to match the code. If you prefer a different name, update the `href` in `Hero.tsx` accordingly.
